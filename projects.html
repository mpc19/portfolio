Projects Overview

Here you can find some projects I have worked on, these are my publicly available projects that you can find online or hosted via GitHub. The projects are mostly coursework completed through the help of Correlation One course while others are projects I've done to test my skills and knowledge including creation of a music player designed to prevent the repetition of music over a set time period and its website.

Weather App

A weather app which utilises the OpenWeather API to provide accurate information about the weather of a specified location and a associated picture using the Unsplash API

Technologies used: HTML, JS, CSS

React To-do List

My first project in React which is a to-do list with CSS animations to add some dynamicism to it.


Technologies used: React, JS, HTML, CSS

Dropbox

Dropbox was my college coursework for Game Design, it is a singleplayer endless platformer game built in C# with exponential difficulty increases


Technologies used: C#, WinForms

Portfolio

The site you are on now which I use as my portfolio of my skills and experience.

Coast

Coast is a customisable, lightweight music player designed to avoid the annoying problem that plagues seemingly every audio source whether radio or streaming services; repetition of music.


Technologies used: C#, HTML, CSS, Bootstrap

VirusTotal Demo

A demonstration project of utilising BunifuFramework and APIs to create a useful program to check and scan files to see if they are malicious or not.


Technologies used: C#, .NET, WinForms
